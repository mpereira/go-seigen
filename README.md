## Dependencies

- [PureScript](http://www.purescript.org/download/)
- [pulp](https://www.npmjs.com/package/pulp)

## Install

```Bash
git clone https://github.com/mpereira/go-seigen.git
cd go-seigen
pulp build
```

## Run

```Bash
pulp run
```
